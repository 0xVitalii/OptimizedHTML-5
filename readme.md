<h1>OptimizedHTML 5</h1>
<p>Lightweight startup HTML5 template, based on Gulp.</p>

<p>
	<img src="https://raw.githubusercontent.com/agragregra/oh5/master/app/img/_src/preview.jpg" alt="Start HTML Template">
</p>

<p>Author: <a href="https://webdesign-master.ru" target="_blank">WebDesign Master</a></p>

<p><strong>OptimizedHTML 5</strong> is lightweight start HTML5 template with Gulp 4, CSS Reboot (Bootstrap reboot), Sass, Browsersync, Autoprefixer, Clean-CSS, Uglify-ES and Rsync support. OptimizedHTML 5 contains a <strong>.htaccess</strong> file with caching rules for web servers.</p>
